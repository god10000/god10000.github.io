毫米波雷达数据帧解析

指令

关闭睡眠监测

53 59 84 00 00 01 00 31 54 43

✅ 求和：

0x53 + 0x59 + 0x84 + 0x00 + 0x00 + 0x01 + 0x00
= 83   + 89   + 132  + 0    + 0    + 1    + 0
= 305

→ 305 = 0x131

我们只取 低8位（即 0x31）

5359 84 00 0001 00 sum 5443
