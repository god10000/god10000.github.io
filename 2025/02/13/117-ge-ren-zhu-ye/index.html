CSS 常用属性

1. 布局

属性 功能 示例
display 设置元素的显示方式 display: flex;
flex 定义 flex 容器内的项目如何分配空间 flex: 1;
flex-direction 设置 flex 容器内子元素的排列方向 flex-direction: column;
justify-content 控制 flex 项目在主轴上的对齐方式 justify-content: center;
align-items 控制 flex 项目在交叉轴上的对齐方式 align-items: flex-start;
align-self 控制单个 flex 项目在交叉轴上的对齐方式 align-self: center;
grid-template-columns 定义 grid 容器的列宽 grid-template-columns: 1fr 2fr;
position 设置元素的定位方式 position: absolute;
top, right, bottom, left 定位元素的位置 top: 10px; left: 20px;
z-index 设置元素的堆叠顺序 z-index: 10;

2. 尺寸与边距

属性 功能 示例
width 设置元素的宽度 width: 100px;
height 设置元素的高度 height: 200px;
min-width 设置元素的最小宽度 min-width: 200px;
min-height 设置元素的最小高度 min-height: 100px;
max-width 设置元素的最大宽度 max-width: 500px;
max-height 设置元素的最大高度 max-height: 500px;
margin 设置元素的外边距（上、右、下、左） margin: 10px 20px 30px 40px;
padding 设置元素的内边距（上、右、下、左） padding: 10px 20px;
box-sizing 控制元素的盒子模型（content-box、border-box） box-sizing: border-box;

3. 背景与颜色

属性 功能 示例
background-color 设置元素的背景颜色 background-color: #f0f0f0;
background-image 设置元素的背景图片 background-image: url('image.jpg');
background-position 设置背景图片的位置 background-position: center;
background-size 设置背景图片的尺寸（cover、contain） background-size: cover;
background-repeat 设置背景图片是否平铺 background-repeat: no-repeat;
color 设置文本的颜色 color: red;
opacity 设置元素的透明度 opacity: 0.5;

4. 文本

属性 功能 示例
font-size 设置字体大小 font-size: 16px;
font-family 设置字体系列 font-family: Arial, sans-serif;
font-weight 设置字体的粗细 font-weight: bold;
line-height 设置文本的行高 line-height: 1.5;
text-align 设置文本的水平对齐方式 text-align: center;
text-transform 控制文本的大小写 text-transform: uppercase;
letter-spacing 设置字符间距 letter-spacing: 1px;
word-spacing 设置单词间距 word-spacing: 2px;
text-decoration 设置文本的修饰 text-decoration: underline;
white-space 设置如何处理空白符 white-space: nowrap;
text-shadow 给文本添加阴影效果 text-shadow: 2px 2px 5px gray;

5. 边框

属性 功能 示例
border 设置边框样式（border-width, border-style, border-color） border: 2px solid red;
border-radius 设置元素的圆角 border-radius: 10px;
border-top 设置上边框 border-top: 3px solid blue;
border-right 设置右边框 border-right: 2px dashed green;
border-bottom 设置下边框 border-bottom: 1px dotted black;
border-left 设置左边框 border-left: 5px solid purple;

6. 盒子模型

属性 功能 示例
box-shadow 给元素添加阴影 box-shadow: 10px 10px 5px gray;
outline 设置元素的轮廓线（不影响布局） outline: 2px solid red;
outline-offset 设置轮廓的偏移量 outline-offset: 5px;
resize 控制元素是否可调整大小 resize: both;

7. 过渡与动画

属性 功能 示例
transition 设置元素的过渡效果 transition: all 0.3s ease;
transition-duration 设置过渡的持续时间 transition-duration: 0.5s;
transition-timing-function 设置过渡的时间函数（ease, linear 等） transition-timing-function: ease-in;
animation 设置元素的动画效果 animation: spin 2s infinite;
keyframes 定义动画的关键帧 @keyframes spin { 0% { transform: rotate(0); } 100% { transform: rotate(360deg); } }

8. 其他

属性 功能 示例
visibility 控制元素的可见性 visibility: hidden;
cursor 设置鼠标光标的样式 cursor: pointer;
filter 应用视觉效果（如模糊、亮度等） filter: blur(5px);
clip-path 定义元素的剪裁路径 clip-path: circle(50%);
object-fit 设置嵌入元素的适应方式 object-fit: cover;

9. 响应式设计

属性 功能 示例
@media 使用媒体查询进行响应式设计 @media (max-width: 768px) { /* CSS */ }
min-width 在媒体查询中设置最小宽度 @media (min-width: 600px) { /* CSS */ }
max-width 在媒体查询中设置最大宽度 @media (max-width: 1200px) { /* CSS */ }

总结：

这个表格涵盖了常用的 CSS 属性，包括布局、尺寸、文本、背景、边框、动画等方面。你可以根据需要查找和使用这些属性来设计和调整网页的样式。

问题

  1. 本应该出右侧的，出现在了下侧

    容器使用flex，父容器中设置flex的垂直排列，改为横向排列flex-direction: row;

  2. 让头像居中

    text-align: center;

  3. 调整左右侧宽度的时候，出现滚动条，且右侧无法调节

  • 在 position: relative 的情况下，left 仅仅是 相对于它原本的位置移动，并不会真正改变布局。

  • right 在 relative 下不会影响 left，只有当 position: absolute 或 fixed 时，right 才会真正起作用。

  • left: 200px; 导致 .main 向右偏移，但宽度不变，所以它可能溢出了视口（浏览器可视区域）。

  • flex 默认让子元素按原始宽度排列，如果 .main 里的子元素宽度超出了 .main 的 width，就会导致溢出，触发水平滚动条。

  • 没有使用 overflow: hidden;：默认情况下，超出容器的内容不会自动隐藏，而是会触发滚动条。
