不对，随机iv再setup里面，所以是因为收发不同步的问题，导致有一部分数据永远也接收不到

问题点

串口乱码

你在串口中这样打印：

Serial.printf("📡 发送段 [%04X]: %s\n", segmentID, mData.c_str());

但 mData 是原始二进制数据（密文），它可能包含：

  • 控制字符（如 0x01, 0x0A, 0x13）
  • UTF-8 非法字节
  • 串口不能解析的转义字符

所以 %s 打印出来就变成乱码或花屏。

请把这一行：

Serial.printf("📡 发送段 [%04X]: %s\n", segmentID, mData.c_str());

替换为：

Serial.printf("📡 发送段 [%04X]: ", segmentID);
for (size_t i = 0; i < mData.length(); i++) {
    Serial.printf("%02X ", (uint8_t)mData[i]);
}
Serial.println();
