kali虚拟机网络配置
2024-01-26 20:36:07 #kali

网络配置

暂时改变ip

在root权限下执行

ifconfig                           #查看本机IP地址
vim /etc/network/interfaces        #修改网卡配置文件

auto eth0
iface eth0 inet dhcp

ifconfig eth0 192.168.31.119/24    #配置临时ip
route add default gw 192.168.31.1  #配置默认路由

/etc/init.d/networking restart     #重启网络服务 
echo name server 8.8.8.8> /etc/resolv.conf 
                                   #配置dns服务器

永久改变ip

vim /etc/network/interfaces #修改网卡配置文件

auto eth0
iface eth0 inet static
address 192.168.31.119
netmask 255.255.255.0
gateway 192.168.31.1

ssh连接虚拟机

netstat -lnt                 //查看开放是否开放22端口

vim /etc/ssh/sshd_config

permitrootlogin yes
pubkeyauthenticatation yes

/etc/init.d/ssh restart

lsof -i :22                 //查看22端口是否被监听

/etc/init.d/ssh status      //查看ssh服务状态，running则表示已启动成功

update-rc.d ssh enable      //设置为自动启动ssh登录服务

error

如果ssh连接不上且ping不通的话,把虚拟机的nat模式换成桥接模式
nat模式下不可以ssh，虚拟机可以上网
桥接模式下可以ssh，但是虚拟机不能上网！！

解决无法上网
vim /etc/resolv.conf

nameserver dns
nameserver 114.114.114.114

即可上网

换源

deb https://mirrors.aliyun.com/kali kali-rolling main non-free contrib
deb-src https://mirrors.aliyun.com/kali kali-rolling main non-free contrib
apt update
