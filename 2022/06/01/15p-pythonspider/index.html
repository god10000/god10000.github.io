学习python爬虫

关于爬虫

  • 1.网络爬虫（又被称为网页蜘蛛，网络机器人）就是通过编写程序让其模拟浏览器向服务器发送网络请求，接收请求响应,是一种按照一定的规则,自动地抓取互联网信息的程序.
  • 2.而浏览器作为客户端从服务器端获取信息，然后将信息解析，并展示给我们
  • 3.模拟：浏览器是一个纯天然的爬虫工具
    浏览器原理：在浏览器中输入一个网址，然后浏览器就会去这个网址所在的服务器要数据，这个过程也叫请求。
  • 4.爬取：
    • 爬取一整张页面数据(通用爬虫)
    • 爬取页面局部数据（聚焦爬虫）
  • 5.爬虫并不是Python独有的，可以做爬虫的语言有很多例如：PHP,JAVA,C#,C++,Python，选择Python做爬虫是因为Python相对来说比较简单，而且功能比较齐全。
    原则上,只要是浏览器(客户端)能做的事情，爬虫都能够做。

爬虫的基础步骤

审查元素

  • 在浏览器的地址栏输入URL地址(就是网址)，在网页处右键单击，找到检查，如下图所示：(不同浏览器的叫法不同，Chrome浏览器叫做检查，Firefox浏览器叫做查看元素，但是功能都是相同的)(快捷键F12)

  • HTML 超文本标记语言 由一个个标签组成,我们想要的数据就在标签里

  • 网页的html代码如图,点击箭头所指的图标,然后将鼠标移动到网页的任意一部分,右侧会出现与之对应的代码,将鼠标移动到代码上同理.

箭头所指的就是标签,class是标签的属性用来区分标签的不同

print(soup.p.text)                    #打印网页中的p标签的内容,会打印出来第一个p标签的数据
print(soup.find('p').text)            #和上一个一样获取第一个符合标签名的内容
print(soup.find('div',class_="content__list--item").text) #精确查找,class后面一定要有_,不能停和python的关键字重名,否则会被当做关键字,报错
print(soup.find_all("div"))#精确查找   #所有div标签
print(soup.fin_all('dib',class_="content__list--item")[1].text)   #findall也可以精确查找,符合条件的第一条数据

写代码前需要掌握的内容

requests库是第三方库，需要我们自己安装。

实例

import requests          #导入requests模块(请求)
#url = ''                #网址
#res=requests.get(url)   #请求
#print(res)              #打印结果为200
#print(res.text)         #查看数据 .text
#HTML 超文本标记语言  由一个个标签组成
import bs4               #导入bs4模块(解析)
#soup=bs4.BeautifulSoup(res.text,'html.parser')  #加工成可以标签定位的数据
import openpyxl          #导入写入excel的模块

wb = openpyxl.Workbook() #创建工作薄
ws = wb.active           #创建工作表

ws.append(["详细信息","优势","价格"])    #写入表头 
for i in range(1,6): #生成一个1-5的序列  [1,2,3,4,5]
    url = "https://bj.lianjia.com/zufang/pg{}/#contentList".format(i)  #在扩号里放入i的值,通过观察发现1-5页的网址的规律
    res= requests.get(url)
    soup = bs4.BeautifulSoup(res.text)
    for i in soup.find_all("div",class_="content__list--item"):
        lst =[]
        lst.append(i.find("p",class_="content__list--item--des").text)
        lst.append(i.find("p",class_="content__list--item--bottom oneline").text)
        lst.append(i.find("span",class_="content__list--item-price").text)
        ws.append(lst)
wb.save("链家租房信息.xlsx")                                            #保存为excel文件
