浏览器脚本 vs. 用户脚本呢

  1. JavaScript 脚本：
    • 执行环境：通常嵌入在网页中，通过 <script> 标签加载，或者在浏览器的控制台中运行。
    • 目的：实现页面功能、处理用户交互、操作 DOM（文档对象模型）等。
  2. 用户脚本：
    • 执行环境：由用户脚本管理器（如 Tampermonkey 或 Greasemonkey）注入到网页中。它们运行在用户的浏览器中，可以修改网页的行为和外观。
    • 目的：用户定制网页的功能，添加额外的功能或修复网页上的问题。

document.addEventListener

document.addEventListener 是 JavaScript 的标准方法，用于在指定的事件发生时执行指定的函数。这是 DOM API 的一部分，允许你在网页上监听和处理用户交互或其他事件。

用法示例：

javascript复制代码document.addEventListener('DOMContentLoaded', function() {
    // 代码在文档完全加载后运行
    console.log('页面加载完成');
});

Node.js 环境

如果你在 Node.js 环境中运行 JavaScript 代码，document 对象是不存在的，因为 Node.js 是一个服务器端的 JavaScript 环境，没有浏览器提供的 DOM API。

解决方法：

  • 如果你需要在 Node.js 中处理网页内容，考虑使用类似 jsdom 的库来模拟浏览器环境。
javascript复制代码const { JSDOM } = require('jsdom');
const dom = new JSDOM('<html><body><h1>Hello world</h1></body></html>');
const document = dom.window.document;

// 现在你可以使用 document 对象
console.log(document.querySelector('h1').textContent); // 输出: Hello world

            }, getRandomDelay());

                mobileField.value = '13304272398';
            }, getRandomDelay());

            // 输入密码
            setTimeout(() => {
                passwordField.value = '522411Wjq.';
            }, getRandomDelay());

            // 检查提交按钮是否存在，并点击提交
            if (submitButton) {
                setTimeout(() => {

            }

百度网盘链接提取

// ==UserScript==
// @name         百度网盘链接及提取码处理
// @namespace    http://tampermonkey.net/
// @version      1.0
// @description  识别并处理百度网盘链接及其提取码，自动填写提取码
// @author       南北的猫
// @match        *://*/*
// @match        *://pan.baidu.com/*
// @grant        none
// ==/UserScript==

(function() {
    'use strict';

    // 在非百度网盘页面上识别链接及提取码
    if (!location.href.includes("pan.baidu.com")) {
        document.addEventListener('mouseup', function(e) {
            let selectedText = window.getSelection().toString();
            let match = selectedText.match(/https?:\/\/pan.baidu.com\/s\/[a-zA-Z0-9_\-]+(\s*提取码:\s*(\w+))?/);

            if (match) {
                let link = match[0];
                let extractCode = match[2];

                if (extractCode) {
                    let confirmJump = confirm(`检测到百度网盘链接：${link}\n提取码：${extractCode}\n是否跳转并尝试使用提取码？`);
                    if (confirmJump) {
                        // 将提取码存储在localStorage，以便在跳转后使用
                        localStorage.setItem('autoFillExtractCode', extractCode);
                        window.open(link, '_blank');
                    }
                } else {
                    if (confirm(`检测到百度网盘链接：${link}\n是否跳转？`)) {
                        window.open(link, '_blank');
                    }
                }
            }
        });
    } else {
        // 在百度网盘页面尝试自动填写提取码
        window.addEventListener('load', function() {
            let autoFillExtractCode = localStorage.getItem('autoFillExtractCode');
            if (autoFillExtractCode) {
                let codeInput = document.querySelector('input selector'); // 更新为实际的提取码输入框选择器
                let submitButton = document.querySelector('button selector'); // 更新为实际的提交按钮选择器

                if (codeInput && submitButton) {
                    codeInput.value = autoFillExtractCode;
                    submitButton.click();
                    localStorage.removeItem('autoFillExtractCode'); // 清理localStorage
                }
            }
        });
    }
})();
