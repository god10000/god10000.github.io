安装V2ray

  • 新 V2Ray 白话文指南(推荐)

  • 官方文档

官方脚本

// 安裝執行檔和 .dat 資料檔
bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh)

移除

bash <(curl -L https://raw.githubusercontent.com/v2fly/fhs-install-v2ray/master/install-release.sh) --remove

也可以拉安装包Releases · v2fly/v2ray-core

#启动 V2Ray:
systemctl start v2ray

#设置开机自启动 V2Ray:
systemctl enable v2ray

#查看运行状态
systemctl status v2ray

v2ray配置格式

sudo vim /usr/local/etc/v2ray/config.json

基础配置: 

{
  "inbounds": [
      {   
         "listen": "[::]",# 监听ipv6端口 
          "port": 10000,  # 服务器监听端口
          "protocol": "vmess",
          "settings": {
              "clients": [
                  {
                      "id": "93d177bc-5d90-0251-69d1-9a30237a9bac",
                      "alterId": 0              #可有可无,额外id
                  }
              ]
          }
      }
  ],
  "outbounds": [
      {
          "protocol": "freedom"
      }
  ]
}
  • 可以正常使用

TLS(暂无法使用)

TLS 中文译名是传输层安全, TLS 是证书认证机制，所以使用 TLS 需要证书

条件:

  • 一个绑定ip的域名
  • 一个证书(我这里使用的证书是cloudflare客户端证书)

证书可以cloudflare生成,也可以*[acme.sh](acmesh-official/acme.sh: A pure Unix shell script implementing ACME client protocol (github.com))*脚本生成

配置:

{
  "inbounds": [
    {
      "listen": "[::]",# 监听ipv6端口 
      "port": 443, # 建议使用 443 端口
      "protocol": "vmess",    
      "settings": {
        "clients": [
          {
            "id": "23ad6b10-8d1a-40f7-8ad0-e3e35cd38297",  
            "alterId": 64
          }
        ]
      },
      "streamSettings": {
        "network": "tcp",
        "security": "tls", # security 要设置为 tls 才会启用 TLS
        "tlsSettings": {
          "certificates": [
            {
              "certificateFile": "/usr/local/etc/v2ray/v2ray.crt", # 证书文件
              "keyFile": "/usr/local/etc/v2ray/v2ray.key" 		# 密钥文件
            }
          ]
        }
      }
    }
  ],
  "outbounds": [
    {
      "protocol": "freedom",
      "settings": {}
    }
  ]
}

服务端和客户端都不报错, 只有测试真延迟的时候,说503错误, 但是无法访问网络
